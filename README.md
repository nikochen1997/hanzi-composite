# 汉字合体字生成器

一个网页应用，用户输入4个汉字，系统将其合成为一个艺术合体字图片。风格为意象化书法（如"一帆风顺"可合成为帆船形态的书法字）。

## 产品功能

- **四字词语合成**：输入4个汉字，生成艺术合体字
- **双版本对比**：每次生成2张不同风格的图片供选择
- **历史记录**：保存所有生成记录，本地存储
- **一键下载**：选择喜欢的版本直接下载保存
- **热门词语快捷入口**：提供常用的吉祥词语快捷输入

## 快速开始

### 1. 安装依赖

```bash
cd 合成字新产品
npm install
```

### 2. 配置API Key

编辑 `.env` 文件，填入你的 Google Gemini API Key：

```bash
# 从 https://aistudio.google.com/app/apikey 获取
GEMINI_API_KEY=your_gemini_api_key_here
```

### 3. 启动服务

```bash
npm start
```

### 4. 访问应用

打开浏览器访问 http://localhost:3000

## 技术架构

### 前端
- 纯静态网页：HTML + CSS + JavaScript
- 移动端优先的响应式设计
- 模拟手机外框的UI设计
- 本地存储历史记录 (localStorage)

### 后端
- Express 轻量框架
- 接收前端请求 → 调用 Gemini API → 返回图片
- API Key 放在后端，不暴露给前端
- CORS 跨域支持

### API
- Google Gemini 2.0 Flash (支持图像生成)
- 并行调用2次生成不同版本
- 返回Base64格式的图片数据

## 核心流程

```
用户输入4个汉字
    ↓
前端发送请求到后端 POST /api/generate
    ↓
后端读取4个规则文档作为上下文
    ↓
后端调用 Gemini API（并行2次）
    ↓
返回2张合体字图片（Base64）
    ↓
前端展示双图，用户选择
    ↓
用户可下载选中的图片
```

## 规则库说明

合体字生成基于4个专业规则库：

1. **拓扑编码库**：定义汉字的"灵魂锚点"和空间象限分布
2. **几何优化库**：处理笔画级别的几何变换和借笔融合算法
3. **风格特征库**：定义毛笔书法的墨迹质感、纸张纹理等艺术参数
4. **语义校验库**：确保合成字语义完整、可识别

## 页面说明

### 合成页（3个状态）

1. **输入状态**：
   - 4个汉字输入框（点击自动弹出键盘）
   - 热门词语快捷入口（药丸形状标签）
   - 最近合成记录横向滚动

2. **生成中状态**：
   - 圆环+方块毛笔动画
   - 4步骤进度指示点
   - 文案轮播提示

3. **结果展示**：
   - 双图并排展示（160x160）
   - 点击选中，带✓标记
   - 保存/重新生成按钮

### 历史页

- 2列网格展示历史记录
- 每张卡片显示缩略图、词语、日期
- 支持删除模式
- "再来一张"快捷重新生成

## 部署建议

### 本地开发
```bash
npm run dev
```

### 生产部署

推荐使用 Railway 或 Render：

1. 推送代码到 GitHub
2. 在 Railway/Render 创建新项目
3. 绑定 GitHub 仓库
4. 设置环境变量 `GEMINI_API_KEY`
5. 自动部署完成

### 环境变量

| 变量名 | 说明 | 必需 |
|--------|------|------|
| GEMINI_API_KEY | Google AI Studio API Key | 是 |
| PORT | 服务器端口，默认3000 | 否 |

## 注意事项

1. **API Key 安全**：`GEMINI_API_KEY` 只能放在后端 `.env` 中，不要提交到Git仓库
2. **并发限制**：Gemini API 有调用频率限制，已做并行2次调用优化
3. **图片大小**：生成的图片为PNG格式，大小约50-100KB
4. **历史记录**：浏览器本地存储最多50条记录
